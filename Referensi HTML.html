Karakteristik

Isi portfolio saya yaitu menunjukan deskripsi singkat mengenai diri saya sendiri, menunjukan skill-skill programming saya beserta project-project yang saya pernah buat, lalu ada bagian kontak agar pembaca bisa menghubungi saya.

Gaya design yang saya pilih yaitu konsep gaya minimalis dan bertema gelap. Karena menurut saya, tema gelap identik dengan programming dan teknologi dan gaya minimalis adalah gaya yang berkesan modern. Gaya design website portfolio saya berdasar pada cita-cita saya sebagai software engineer.

Font yang saya pilih yaitu font sans-serif rounded karena font ini bagus dipandang, tidak terlalu formal bersifat modern.
